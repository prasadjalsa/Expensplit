💡What is ExpenSplit?

ExpenSplit is a group expense tracker — for trips, flatmates, events, or any shared spending. It works like Splitwise but lives in a single HTML file you host on GitHub Pages.

  • No sign-up — access is controlled by a room link
  • Real-time sync — all members see changes instantly via Firebase
  • Flexible splits — equal, unequal, %, or share-based
  • Smart settlements — fewest possible transfers to clear all debts
  • Sign-off flow — each member acknowledges expenses before settling
🔗Rooms & Sharing

A room is a private shared workspace for your group. Every expense, balance, and settlement lives inside one room.

  • A random Room ID is created automatically when you first open the app
  • The Room ID appears in the URL as #room=abc123xyz and on this page
  • Anyone who opens the same URL joins the same live room
  • Data is stored in Firebase Firestore — it persists in the cloud, not just your browser
TipTap the 🔗 button in the header or Copy Link on the People tab to copy your room URL and send it to your group.
ImportantAnyone with the room link can view and edit the data — keep it within your group.
👥How to: Set up a group
  1. Name the group
    Tap "Tap to add a name" in the header — type a name like "Goa Trip 2025" and press Enter. You can also do this via Settings ⚙️.
  2. Add all members
    On the People tab, type each person's name and tap Add. Repeat for every member. Each gets a colour-coded avatar.
  3. Share the room link
    Tap Copy Link on the People tab. Send the URL to your group — they open it and immediately join the same room.
TipAdd everyone before adding expenses — people can only be assigned to expenses if they exist in the group.
🧾How to: Add an expense
  1. Go to Expenses tab and tap +
    The green + button is at the bottom-right of the screen.
  2. Enter description, amount, and date
    The app auto-picks an emoji icon based on keywords (e.g. "dinner" → 🍝). Date defaults to today.
  3. Set who paid
    Select the payer from the dropdown. The hint turns green when payer amounts match the total.
  4. Choose a split type
    Pick Equal, Unequal, %, or Shares and configure the amounts (see Split Types below).
  5. Tap Save Expense
    The expense appears instantly for all group members.
TipTap any expense card to view a full breakdown. From the detail view you can Edit or Delete it.

💳How to: Multiple payers

Use this when two or more people split a bill at the counter.

  1. Select the first payer and enter their amount
  2. Tap "+ Add payer" for a second row
    Select the second person and enter what they paid.
  3. Confirm the hint is green
    Payer total must equal the expense total before you can save.
✅How to: Sign off

Before settling, each member confirms the expense list is complete and correct. This prevents disputes.

  1. Go to the Settle Up tab
    The Sign-offs card appears at the top with each member's status.
  2. Each member taps their own Confirm button
    Records the date of confirmation. Badge turns ✓ Confirmed.
  3. Once all confirmed, proceed to settle
    Summary shows "✓ All N members confirmed".

Sign-off statuses:

Pending — not confirmed yet.

✓ Confirmed — confirmed, no new expenses since.

⚠ N new expenses — confirmed earlier but new expenses were added after. Tap Re-confirm.

💸How to: Record a payment

The Settle Up tab shows the minimum number of transfers needed to clear all balances.

  1. Review Suggested Settlements
    Each row shows who pays whom and how much.
  2. Make the payment outside the app
    UPI, bank transfer, cash — any method works.
  3. Tap Settle on that row
    Pre-filled dialog opens. Adjust amount or add a note (e.g. "GPay"), then tap Confirm Payment.
  4. Repeat until all transfers are recorded
    App shows 🎉 All settled up! when every balance is cleared.
TipThe smart settlement algorithm minimises transfers. 5 people with complex debts may need only 4 (or fewer) payments to fully balance out.
⚙️How to: Settings
Change group name

Tap "Tap to add a name" in the header to edit inline, or open Settings ⚙️ → Group / Expense Name.

Change currency symbol

Open Settings ⚙️ → Currency Symbol. Type your symbol (e.g. $ € £ ¥) and tap Save. All amounts update immediately.

Remove a member

On the People tab, tap ✕ next to the person's name. If they appear in any expense you'll be asked to confirm.

🔄How to: Reuse for a new trip
  1. Record all payments first
    Go to Settle Up and tap Settle on every suggested transfer to preserve the history.
  2. Open Settings ⚙️ and tap Clear All Data
    Expenses and settlements are deleted. Group name, members, and currency are kept.
  3. Update the group name
    Tap the name in the header and type the new trip name.
  4. Start adding new expenses
    Same room ID — no need to re-share the link.
❓FAQ
Is my data private?

Data is secured by obscurity — only people with the room link can access it. The Room ID is a random 12-character string that is practically impossible to guess. Don't share the link publicly.

What if I lose the room link?

The Room ID is saved in your browser's local storage. Opening the app on the same browser/device reconnects automatically. The Room ID is also shown on the People tab — copy it and store it safely. On a new device, ask a group member with the link to share it again.

Can multiple people add expenses at the same time?

Yes. Firebase Firestore syncs in real time. When any member saves an expense, all other open browsers update within seconds. The ● Live pill confirms the connection is active.

What does ● Live / ● Offline mean?

● Live — connected to Firebase, changes sync in real time.

● Offline — Firebase connection failed. The app shows the last saved state from your browser. Changes won't sync until reconnected.

How does the smart settlement work?

The app calculates each person's net balance (total paid minus total owed). It then uses a debt-minimisation algorithm to find the fewest transfers that bring every balance to zero — significantly fewer than settling every pair individually.

Can I edit or delete an expense after saving?

Yes. Tap the expense card to open its detail view, then tap Edit or Delete. Adding or deleting expenses marks previously confirmed sign-offs as needing re-confirmation.

What does "Re-confirm needed" mean?

A member confirmed the expense list earlier, but new expenses were added (or deleted) after that. The ⚠ N new expenses badge prompts them to review and tap Re-confirm if they agree.

Does the app work without internet?

Partially. Existing data loads from browser local storage when offline. Changes won't sync until reconnected. Firebase may also buffer writes locally and replay them when the connection is restored.

What happens when I tap Clear All Data?

All expenses, settlements, and sign-offs are permanently deleted from both the browser and Firebase. Group name, members, and currency are preserved. This cannot be undone.

Can I use this for multiple trips at once?

Yes — each unique room link is a separate workspace. Open the app in a fresh browser tab (without a room hash in the URL) to generate a new room, then share that link for a different group or trip.

The % input keeps unchecking the person — what do I do?

Tap directly on the number input box, not on the row background. The row background acts as a checkbox toggle. Targeting the input itself avoids accidentally deselecting the person.

Can I change the currency mid-trip?

Yes — open Settings ⚙️ and update the currency symbol. This is a display-only change; underlying numbers are not converted. If your group used two currencies, convert amounts manually before entering.
